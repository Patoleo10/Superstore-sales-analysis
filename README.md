# Superstore Sales Analysis

## Project Overview
Analysis of US Superstore sales data using SQL in Mode Analytics 
to uncover revenue trends, top products, and regional performance.

## Tools Used
- SQL (PostgreSQL)
- Mode Analytics

## Dataset
Superstore Sales Dataset from Kaggle — 9,800 rows of US retail 
sales data covering 2015–2018.

## Key Questions Answered
1. Which product category generates the most revenue?
2. What are the top 5 best-selling products?
3. Which region has the highest number of orders?
4. How does sales trend across the months of the year?

## Key Findings
- Technology is the top revenue category at $827,455
- Canon imageCLASS 2200 Copier is the best-selling product at $61,599
- The West region leads with 3,140 orders
- Sales peak in November ($350,161) and dip lowest in February ($59,371)
